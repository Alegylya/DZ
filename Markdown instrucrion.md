# Инструкция для работы с Makdown

## Выделения статуса

Чтобы выделить текст курсовм необходимо обратить его  звездочками (*). Например, *вот  так*.

Чтобы выделить текст полужирным, необходимо обрамить его двойными звездочками (**). 
Например, **вот так**
